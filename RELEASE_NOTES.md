# Release Notes

## 6.0.0

本版本完成了从旧版 Windows Python 工程到全 Rust + Tauri 2 终局工程的重构，核心变化如下：

- 建立 Rust workspace，拆分为领域、平台、存储、捕获、检测、输入、运行时和诊断模块。
- 统一 Tauri 命令层、托盘、能力模型和运行时控制器，移除旧版 sidecar 与多条并行运行路径。
- 重建企业控制台前端，覆盖总览、任务配置、模板库、目标窗口、实时预览、日志诊断和系统设置。
- 建立工作区单元测试、Windows 集成测试分组、前端冒烟测试、诊断包导出和性能预算基线。
- 清理旧版 Python 原始工程目录，保留最小旧版导入夹具用于迁移验证。
- 完成发版前隐私排查，移除源码与测试中的绝对示例路径，避免将本机路径样式内容带入远端仓库。
